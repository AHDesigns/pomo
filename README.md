# Usage
<!-- usage -->
```sh-session
$ npm install -g @ahdesigns/pomo
$ pomo COMMAND
running command...
$ pomo (-v|--version|version)
@ahdesigns/pomo/1.0.3 darwin-x64 node-v12.4.0
$ pomo --help [COMMAND]
USAGE
  $ pomo COMMAND
...
```
